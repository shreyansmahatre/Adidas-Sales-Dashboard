# Adidas-Sales-Dashboard
# Sales & Profit Analysis Dashboard – Power BI

## Project Overview

This project is an interactive Power BI dashboard designed to analyze sales and profit performance using business intelligence and data visualization techniques. The dashboard helps users monitor key performance indicators (KPIs), identify sales trends, evaluate profitability, and generate actionable business insights.

The project focuses on transforming raw business data into meaningful visual reports that support data-driven decision-making.

---

# Objectives

* Analyze overall sales and profit performance
* Identify high-performing products and regions
* Track KPIs and business growth trends
* Generate insights for strategic decision-making
* Improve reporting through interactive visualizations

---

# Tools & Technologies Used

* Power BI
* Power Query
* DAX (Data Analysis Expressions)
* Excel / CSV datasets
* Data Cleaning & Transformation
